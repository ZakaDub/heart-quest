# Heart Quest — мини-игра-квест

Локальный веб-квест из 3 головоломок + финал. HTML, CSS, JavaScript — без библиотек и backend.

## Запуск

Открой `index.html` в браузере (двойной клик или Live Server).

## Структура

```
heart-quest/
├── index.html      — страница «Сердце»
├── puzzle2.html    — «Соединение воспоминаний»
├── puzzle3.html    — «Пиксельная раскраска»
├── final.html      — финал
├── style.css       — общие стили
└── script.js       — логика всех страниц
```

## Настройка

**Страница 1:** секретный шарик-кнопка выбирается случайно при каждой перезагрузке.

**Страница 2:** свои фразы в `script.js` → `CUSTOM_PAIRS` (одинаковый `id` = пара). Карточки перетасовываются при каждой перезагрузке.

**Страница 3:** пазл 4×6 с картинкой `assets/meowl-puzzle.png`. Пустая клетка — справа снизу за сеткой.

**Финал** (`initFinalPage()`):

- `CONGRATULATION_MESSAGE` — главный текст поздравления
- `MEDIA_SRC` — путь к фото (`assets/photo.jpg`) или видео
- `MUSIC_SRC` — путь к фоновой музыке (включается только по кнопке)

## Звуки

Генерируются через Web Audio API — не нужны отдельные аудиофайлы.
